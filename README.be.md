# Totus Tuus

**Беларуская** · [English](README.md)

Каталіцкае прыкладанне на беларускай мове: малітоўнік, спеўнік, Святое Пісанне ў беларускіх перакладах і літургічны каляндар — у адным месцы, анлайн і афлайн.

- **Вэб-дадатак:** [app.kasciolhomiel.by](https://app.kasciolhomiel.by)
- **Android:** [Google Play](https://play.google.com/store/apps/details?id=by.totustuus.app)

## Магчымасці

| Раздзел | Апісанне |
|--------|----------|
| **Малітоўнік** | Структураваны збор малітваў з пошукам і закладкамі |
| **Спеўнік і кантарал** | Літургічныя спевы і кантарал |
| **Святое Пісанне** | Некалькі беларускіх (і сумежных) перакладаў Бібліі, пошук па главах, план чытання |
| **Літургічны каляндар** | Штодзённы літургічны кантэкст па дыяцэзіях |
| **Лекцыянарый** | Чытанні на абраны дзень |
| **Урачыстасці і святы** | Галоўныя ўрачыстасці і святыя |
| **Ordo Missae** | Парадак Імшы |

Android-дадатак кэшуе кантэнт для афлайн-выкарыстання. Вэб-дадатак працуе ў браўзеры без усталёўкі (падтрымка PWA).

## Структура рэпазіторыя

```
Totus Tuus/
├── Totus Tuus App/     # Android-дадатак (Kotlin)
├── Web App/            # Кліент для браўзера (HTML, JS, Tailwind CSS)
├── Web Panel/          # PHP-панэль адміністравання і публічнае API
└── totus-app-version.properties   # Агульная версія і ключ API
```

## Патрабаванні

### Android-дадатак

- Android Studio з JDK 17+
- Android SDK 36 (compile), min SDK 24
- Файл `totus-app-version.properties` у карані рэпазіторыя (глядзі існуючы файл як узор)
- Дадаткова: `Totus Tuus App/local.properties` з `totus.publicApiKey=…`, калі не выкарыстоўваеце агульны файл уласцівасцяў

Адкрыйце папку `Totus Tuus App` у Android Studio і запусціце модуль `app`.

### Вэб-дадатак

- Любы статычны вэб-сервер (Apache, nginx і г.д.)
- Для сінхранізацыі кантэнту: працуючае API **Web Panel** (глядзі ніжэй)
- Наладзьце `Web App/api-config.js` (`apiBaseUrl`, `webPanelRootUrl`, `useServerProxy`)
- Для API-проксі: скапіруйце `Web App/api/proxy-secrets.example.php` у `proxy-secrets.php` і ўкажыце ўліковыя дадзеныя

Пасля змены `versionName` у `totus-app-version.properties` абнавіце `meta name="totus-web-build"` і параметры `?v=` у `Web App/index.html` на тую ж версію.

### Web Panel (бэкенд)

- PHP 8+ з PDO/MySQL
- База дадзеных MySQL/MariaDB
- Наладзьце доступ да БД у `Web Panel/api/db.php`
- Задайце публічны ключ API праз зменную асяроддзя `TOTUS_PUBLIC_API_KEY` або `publicApiKey` у `totus-app-version.properties` (павінен супадаць з Android-дадаткам і вэб-проксі)

Панэль адміністравання кіруе малітвамі, Пісаннем, літургіяй, лекцыянарыем, спеўнікам, аб’яўленнямі і звязаным кантэнтам.
